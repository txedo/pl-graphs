Salida de errores

 Top  Previous  Next

Durante el proceso de compilación se pueden generar dos tipos de mensajes: errores y warnings. Los errores pueden ser léxicos (palabras mal formadas), sintácticos (frases mal formadas) o semánticos (identificadores repetidos, etc.).

 

Si se produce algún error durante la compilación del texto de entrada, éste aparecerá en la consola de realimentación y se parará la compilación del grafo en curso, continuando con la compilación del resto de grafos del fichero.

 

Si se produce algún warning, éste no deberá ser ignorado y se recomienda al usuario que preste especial atención ya que puede ser el motivo de un error generado más adelante.

 

Para obtener ejemplos detallados de este tipo de situaciones, se invita al lector a visitar la sección Ejemplos de uso.
