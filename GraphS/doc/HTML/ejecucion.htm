Ejecución

 Top  Previous  Next

La aplicación se distribuye en un archivo ejecutable con extensión JAR. Para un correcto funcionamiento es necesario que en el mismo directorio donde se encuentra el ejecutable exista un directorio llamado reports, un directorio llamado lib que contiene las librerías utilizadas y un últmo directorio llamado UserManual que contiene el manual de ayuda. Todos estos directorios se distribuyen también con la aplicación así que se recomienda no eliminarlos ni renombrarlos.

 

La ejecución de la aplicación puede realizarse de dos formas distintas:

1.Haciendo doble click sobre el icono del fichero GraphS.jar
2.Mediante un terminal o consola, situándonos en el directorio donde se encuentra la aplicación y ejecutando la setencia "java -jar GraphS.jar".

 

Los usuarios de sistemas basados en MS Windows pueden utilizar cualquiera de las dos alternativas mostradas. Sin embargo, los usuarios de sistemas basados en GNU/Linux deberán hacerlo según el método segundo para que la máquina virtual de Java detecte bien el directorio de trabajo y no tenga dificultades en encontrar el directorio reports.
