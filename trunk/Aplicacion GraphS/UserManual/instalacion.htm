Instalación

 Top  Previous  Next

La aplicación está empaquetada en un archivo con extensión .jar autoejecutable, aunque al estar implementada en JAVA se necesita tener instalado el entorno de ejecución de Java (Java Runtime Enviroment) en una versión 1.6.x o superior. Ésta se puede descargar desde la siguiente página:

 

http://java.sun.com/javase/downloads/index.jsp

 

Además de lo anterior, la aplicación hace uso de un programa externo para generar los grafos en imágenes con extensión .png.

Este programa externo se llama GraphViz, que consta de una serie de librerías y aplicaciones. Se puede descargar desde la siguiente página:

 

http://www.graphviz.org/Download..php

 

Una vez descargado el instalador, se debe proceder a instalarlo. Es importante recordar la ruta completa donde se intala, ya que el siguiente paso será almacenarla dentro de la variable de entorno $CLASSPATH para poder ejecutar la aplicación GraphS desde cualquier ubicación (ver Figuras 3 y 4).
