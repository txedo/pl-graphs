Instalación

 Top  Previous  Next

La aplicación está implementada en Java y se distribuye empaquetada en un archivo ejecutable con extensión JAR. Por lo tanto, es necesario que el usuario tenga instalado en su sistema operativo una versión de la máquina virtual de Java (JRE, de sus siglas en inglés Java Runtime Environment) igual o superior a la 1.6.x, que puede descargarse de la siguiente página:

 

http://java.sun.com/javase/downloads/index.jsp

 

Dado el lenguaje de implementación, la aplicación puede ejecutarse tanto en entornos Windows como en entornos basados en Unix (p.e. GNU/Linux). Independientemente del entorno, la aplicación hace uso de un programa externo para generar las imágenes de grafos con extensión PNG. Este programa externo se llama GraphViz, que consta de una serie de librerías y aplicaciones. Se puede descargar desde la siguiente página:

 

http://www.graphviz.org/Download.php

 

Los usuarios de una distribución GNU/Linux basada en Debian también pueden usar el gestor de paquetes e instalar automáticamente el paquete llamado graphviz.
