Compilación

 Top  Previous  Next

Cuando el usuario escribe y abre un fichero GRAPH, puede proceder a la compilación de éste. Para ello, debe pulsa el icono del PLAY (ver Figura 8). Si no existe ningún tipo de error, ya sea léxico, semántico o sintáctico, se generará un fichero HTML en el directorio reports junto con una serie de imágenes, donde cada una corresponde a un grafo o a una operación de un grafo.

 

 

Figura 8 - Compilación

 

 

Si se produce algún tipo de error, será mostrado en la ventana de realimentación. Se recomienda visitar la sección Salida de errores para obtener una información más detallada acerca de éstos.
