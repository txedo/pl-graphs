Graphs es un lenguaje de especificación de grafos que se está diseñando para las prácticas de Procesadores de Lenguajes (ESI@UCLM) para el curso 2008/2009

Autores:
	Jose Domingo López López
	Ángel Escribano Santamarina


En los directorios que comienzan por una numeración del 1 al 5, se puede observar el trabajo realizado a lo largo de las sesiones de prácticas.

En el directorio DOC se pueden encontrar algunos documentos interesantes junto con el instalador del compilador de ficheros DOT (graphviz-2.22.2.msi), necesario para utilizar la aplicación GraphS.

El directorio GraphS contiene todos los ficheros relativos al desarrollo del proyecto en Netbeans, así como documentación final y código fuente.

El directorio Aplicación GraphS contiene las últimas versiones estables tanto del ejecutable, como de la documentación y el manual de usuario.
